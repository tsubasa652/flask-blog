# Flask-Blog

Minimal blog with Flask

## Requirement

Python >= 3.7

## Usage

### Clone

```bash
git clone https://github.com/tsubasa652/flask-blog.git
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run Flask-Blog

```bash
python main.py
```
